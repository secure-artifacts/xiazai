青苹果下载器 · 首次配置

只需配置一次，以后框选链接就能下载。

准备：使用能打开目标表格和视频的 Google 账号。视频仍受 Drive 原有权限控制，软件不会绕过禁止下载设置。
1

创建或选择 Google Cloud 项目

进入 Google Cloud Console，创建新项目或选择团队已有项目。

2

启用两个 API

在“API 和服务 → 库”中分别启用 Google Drive API 与 Google Sheets API。

Drive API 官方快速入门 · Sheets API 官方快速入门

3

配置 OAuth 同意屏幕

在“Google Auth Platform / OAuth 同意屏幕”中填写应用名称和联系邮箱。团队属于同一 Google Workspace 时可按管理员政策选择内部应用；个人账号通常选择外部应用。

如果应用仍处于测试状态，把实际下载者的 Google 邮箱加入测试用户。

4

下载“桌面应用”OAuth JSON

打开“凭据 → 创建凭据 → OAuth 客户端 ID”，应用类型选择 桌面应用，创建后下载 JSON。

官方步骤：创建桌面应用凭据。请不要把这个 JSON 发给无关人员。

5

安装表格端脚本

在目标表格中选择“扩展程序 → Apps Script”，把随软件提供的 Code.gs 全部复制进去并保存，然后刷新表格。

菜单栏会出现 🍏 成品下载。脚本只把框选链接写入当前表格的隐藏队列，不读取或下载 Drive 视频。

6

在软件中完成连接

  1. 打开青苹果下载器 →“设置”。
  2. 选择第 4 步下载的 OAuth JSON。
  3. 粘贴目标 Google 表格完整链接。
  4. 选择下载目录；并发保持建议值 2。
  5. 保存后点击“连接 Google 账号”，在浏览器完成授权。
7

日常使用

框选任意数量的超链接单元格 →“🍏 成品下载”→“成品下载”。右下角会静默显示进度；保持 Windows 客户端打开，任务会自动进入按时间命名的新文件夹。

ZIP 默认关闭更快。需要额外压缩包时，再到设置中打开 ZIP 开关；程序会保留原视频并额外生成压缩包。

手动粘贴文字超链接：在表格中复制目标单元格，切换到软件“粘贴链接”页后直接按 Ctrl+V。软件会优先读取剪贴板中的 HTML/RTF 信息并显示真实 Drive 地址；也可以点击“从剪贴板提取超链接”。

常见问题

  • 403：确认两个 API 已启用、登录账号有文件权限、文件拥有者没有禁止下载。
  • 表格没有菜单：保存 Apps Script 后刷新整个表格页面。
  • 只下载两个？不是。2 是同时下载数；框选 10 个会分批并发并全部完成。
  • 下载中断：保留文件夹中的 .part 文件，再次下载同名文件会继续传输。
