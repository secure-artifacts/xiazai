# 表格端脚本安装

1. 打开需要框选下载的 Google 表格。
2. 选择“扩展程序”→“Apps Script”。
3. 删除编辑器里的示例内容，把 `Code.gs` 全部复制进去并保存。
4. 回到表格并刷新页面。
5. 菜单栏会出现“🍏 成品下载”。
6. 第一次选择“成品下载”时，按 Google 提示授权脚本访问当前表格。

之后只需框选含 Drive 超链接的单元格，选择“🍏 成品下载”→“成品下载”。脚本会在右下角静默显示执行进度，Windows 端打开后自动开始下载。

如果表格已有自己的“手动菜单”，不要保留第二个 `onOpen()`。只需在原有菜单链中加入：

```javascript
.addItem('🍏 成品下载', 'runGreenAppleDownload')
```

一个 Apps Script 项目只保留一个 `onOpen()` 即可，其余青苹果函数保持不变。

支持：

- 单元格显示名字、底层是超链接的富文本（用户截图中的形式）
- `HYPERLINK()` 公式
- 单元格直接显示的 Drive 链接或文件 ID
- 多区域框选与批内自动去重

脚本本身不下载视频，也不要求 Drive 权限；它只在当前表格中维护隐藏队列 `_青苹果下载队列`。
